# House Prices - Advanced Regression Techniques

This repository contains my complete solution for the Kaggle competition **House Prices - Advanced Regression Techniques**.

The goal of the competition is to predict the final sale price of residential homes in Ames, Iowa, using structured tabular data.  
The project includes a complete machine learning workflow: exploratory data analysis, preprocessing, feature engineering, model training, ensemble learning, and final Kaggle submission.

---

## Competition

**Kaggle Competition:** House Prices - Advanced Regression Techniques  
**Task:** Predict house sale prices using regression models  
**Evaluation Metric:** Root Mean Squared Logarithmic Error, RMSLE

---

## Best Kaggle Result

My best submission achieved:

- **Score:** `0.11705`
- **Rank:** `90`

This result was obtained using a combination of strong preprocessing, feature engineering, and ensemble learning with several regression models.
